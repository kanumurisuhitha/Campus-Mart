# Campus Mart

Campus Mart is a GUI-based application built using Python and Tkinter for buying and selling items. It provides users with a platform to register, log in, and perform various actions like browsing items by category, buying items, or adding items for sale.

## Features

- *User Authentication*: Users can register with a username, password, and email, and then log in securely to access the application.

- *Buying Items*: Users can browse items available for sale categorized by type and purchase items of interest. Confirmation prompts ensure secure transactions.

- *Selling Items*: Sellers can add items for sale by specifying item details such as name, price, description, and contact email. Items are categorized for easy browsing by buyers.

## Installation

1. Clone the repository:

   bash
   git clone https://github.com/Susmitha666/Wise_CampusMart.git
2. Install the required dependencies:

   bash
   pip install -r requirements.txt
   

3. Set up the MySQL database:
   - Create two databases: unitrade and categories.
   - Configure the database connection parameters in the code (app.py) according to your MySQL setup.

## Usage

1. Run the application:

   bash
   python app.py
   

2. The application window will open, allowing users to register, log in, and choose between buying and selling items.
